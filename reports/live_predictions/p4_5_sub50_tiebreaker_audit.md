# P4.5 Sub-50 Weak-Home Tiebreaker 病理审计与 PR 设计

## 1. 背景与目标
在最新的实盘复盘中，系统对“赫罗纳 vs 皇家社会”给出了 2-1（主胜）的预测，但实际赛果为 1-1。核心数据表明主胜概率（约 48%）低于平+负概率之和，且 1-1 与 2-1 同赔、概率极度接近并列第一。在此均势结构中，AI 疑似被“主队保级战意”等主观叙事带偏，强行推荐了主胜。

本报告旨在不修改代码、不影响当前线上模型的前提下，通过数据切片审计该现象，评估其是否为系统性问题，并设计合理的 PR（不强制接管比分，而是增强提示和分支策略）。

## 2. 赫罗纳个案切片
- **home_win_pct**: 48.0%
- **draw_pct**: 25.0%
- **away_win_pct**: 27.0%
- **predicted_score**: 2-1
- **top_score_candidates**: [["2-1", 15.0], ["1-1", 15.0], ["1-2", 11.0]]
- **ai_native_reason**: “结合主队保级战意极强，2-1是比1-1更符合基本面逻辑的选择...”

### 审计发现：
1. **主胜概率确实 < 50%**，且 `draw + away` (52%) > `home`。
2. **1-1 与 2-1 概率完全并列 (15.0%)**。
3. **存在 Narrative Override**：AI 明确在分析中承认数据势均力敌，但出于对“主队保级战意”的主观解读，人为选择了 2-1。
4. **尾部预警已生效**：系统已成功打上 `weak_home_favorite_btts_tail` 标签，并且 `risk_level` 被判定为 `high`，前端仪表盘中必然显示为高危。

## 3. 全量同类场景扫描
扫描 `live_predictions_9999_github_actions.json`，寻找所有 `final_direction=home`、`home_win_pct < 50%`、且选择了窄胜主胜（如 1-0/2-1）且存在平局拉力的场次。

**共发现 3 场同类候选：**
1. **周三003 比利亚雷 vs 塞维利亚** (预测: 2-1, 实际: 2-3) - 主胜 49.0%
2. **周三010 赫塔费 vs 马洛卡** (预测: 1-0, 实际: 3-1) - 主胜 41.0% (Matrix已冲突建议1-1)
3. **周四004 赫罗纳 vs 皇家社会** (预测: 2-1, 实际: 1-1) - 主胜 48.0%

可见，这种“主胜不到 50% 但仍因为某些基本面因素死撑一球小胜”的模式，在西甲（高平局联赛）极为常见，且实盘打出的方差极大。

## 4. PR 方案设计与建议
基于以上病理特征，**不建议硬性锁死 1-1（强制改比分）**，因为战意叙事有时确实是敏锐的（如赫塔费 3-1）。正确的做法是把这种“结构性高危与主观叙事冲突”透明化。

### 建议采取的方案：
1. **No-Bet Gate / 风险锁 (推荐)**
   增加 `sub50_tiebreaker_warning` 和 `no_bet_reason` 字段。当前系统将其放在 `recommendation_tier: C`，我们可以在 PR 中升级这一逻辑：当出现 Sub-50 且有同分平局候选时，在 recommendation 层输出明确的 `hedge_recommendation: "主胜概率劣势，建议防 1-1 或放弃"`。
2. **Score Cluster 输出 (推荐)**
   与其纠结单比分，不如输出一个方向锁定的比分簇。将 `score_cluster: ["2-1", "1-1"]` 作为一级字段暴露，明确告诉用户这是一个“主胜与平局的二元博弈”。

### 不建议的方案：
- **硬锁 1-1**：剥夺 AI 的综合判断力，会导致遇到真实战意碾压局时僵化。
- **改 final**：违反了我们确立的“外置风控不干预最终裁判裁决”原则。

## 5. 结论
- **是否建议开 PR**：是
- **PR 建议名称**：`feat: sub-50 weak-home structural risk transparency`
- **PR 范围**：仅修改 `predict.py` 中后处理的字段装饰（生成额外的 warning 和 cluster 字段），不动 `final_direction` 和 `predicted_score`。
