# 阶段1交付报告 — 分位基线 + 线移动方向基线 (2026-05-31)

## 数据家底盘点
- football-data 五大联赛: 1674场, 含**开盘价vs收盘价**(B365/B365C 的 over2.5、亚盘、1X2)
  → 破解"漂移数据抓不到"硬伤: 大小球线移动方向可回测.
  → 但口径是 over/under 2.5, **非竞彩 a4/a5**; 且不覆盖挪瑞芬沙MLS.
- matched_samples 175场: 竞彩 a4/a5 静态快照, 覆盖主力小联赛, 但单联赛样本薄.

## 核心结论1: 大小球线移动方向 = 有效且独立的信号 (STUDY D, 1674场)
| steam(收盘-开盘大球概率) | 真实over2.5 | >=4球 |
|---|---|---|
| 强烈下行<=-5% | 44.0% | 27.5% |
| 基本不动±1.5% | 53.0% | 30.8% |
| 强烈上行>=5% | 63.7% | 37.4% |
- 完全单调(44→64%), 无噪声跳跃.
- 独立信息: 控制收盘价五五开时, 仅凭方向仍分离+3.3pp.
- 巴黎式(线下行+收盘偏小): 真实小球率57.2% vs 基础46.8% (+10.4pp). 用户读法实证成立.

## 核心结论2: 线移动信号强度**因联赛而异**(决定prompt分层)
| 联赛 | 基础over2.5 | 移动方向分离度 |
|---|---|---|
| 德甲 | 63.6% | **+2.6pp 弱** |
| 意甲 | 45.8% | +5.6pp |
| 西甲 | 50.3% | +9.4pp |
| 英超 | 55.0% | +9.7pp |
| 法甲 | 52.9% | **+15.1pp 强** |
→ 大球联赛(德甲)开盘已打满大球预期, 线移动无信息, 应靠绝对大球先验.
→ 中小球联赛(法/英/西)盘面可被资金推动, 线移动含金量高, 应重看漂移方向.
→ **prompt 必须按联赛风格给"线移动"分层赋权, 禁止全局一刀切.**

## 核心结论3: 联赛 a4 基线天差地别 (STUDY C)
德甲中位4.20 / 荷甲4.33 vs 意甲5.30 / 西甲5.20 / 解放者5.50.
→ "a4<=X一刀切"在德甲误报、在意甲漏报. 须用相对联赛分位.
→ 单联赛样本薄(德甲20/意甲23/余个位数)致分位噪声大 → 仍需补厚(对小联赛).

## 核心结论4: 判大球看曲线塌缩不看单点 (用户洞察实证)
a4压缩时a5/a6同步塌缩, 真实进球整体上移. a5/a4斜率<=1.3→66.7%大球.
5.3=排除线(a4>5.3真实大球仅12.8%), 非纳入线.

## 阶段1定调(已确定, 不再回退)
