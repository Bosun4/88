# 读盘 Prompt 重构设计稿 v1 (2026-05-31)
> 目标: 把 AI 从"抄最低赔率比分"改造成"两段式读盘". 全部基于今日 1674+175 场实证.
> 此稿为设计, 落地 predict.py 前须: 备份→本地全回归→前后对比→批准→新分支.

## 一、决策结构: 强制两段式 (核心)
**第一段 — 只定档位, 禁止报比分:**
输出 `goal_tier` ∈ {LOW(0-1球) / MID(2-3球) / HIGH(>=4球)} + `tier_confidence`.
定档依据按下方"读盘三闸"推理, 不准跳过直接写比分.

**第二段 — 档内选比分:**
- 默认落该档**众数比分**: LOW→1-0 / MID→1-1 / HIGH→不押精确(只押大小球方向).
- 选**非众数比分**必须在 `deviation_reason` 写出可证伪的背离理由; 写不出→回落众数 + confidence 下调.
- **0-0 与 1-1 严禁同台比价** (0-0属LOW, 1-1属MID): 先定LOW/MID, 再档内选.

## 二、读盘三闸 (定档位的推理链, 替代旧评分规则)

### 闸1 — 大小球线移动方向 (一级信号, 但分联赛赋权)
看 `ttg_change`/盘口位移: 线下行=往小球, 线上行=往大球.
实证(1674场): 强烈下行→真实over2.5仅44%, 强烈上行→64%, 单调.
**联赛赋权 (关键, 禁止一刀切):**
- 法甲/英超/西甲/意甲(中小球, 盘面可被资金推动): 线移动是**强信号**(+9~15pp), 重看.
- 德甲及大球联赛(挪/瑞/芬/荷/沙/MLS): 开盘已打满大球预期, 线移动**弱信号**(德甲+2.6pp),
  改靠"绝对大球先验"(本就该往大球档偏), 别把线没动当小球证据.

### 闸2 — 大球判据: 看曲线塌缩, 不看单点 (治单点诱盘)
**不要用 a4<=5.3 单点判大球** (实证 a4<=5.3 真实大球仅37.5%, 接近基础32%, 几乎无效).
- 真信号 = 整条大球曲线塌缩: `a5/a4 斜率`. a5/a4<=1.3→66.7%大球; <=1.45→51.6%.
  (a4被压但a5没跟着压=单点诱盘, 是假信号, 不进HIGH档)
- **相对联赛分位**叠加: 同样a4, 德甲(中位4.20)是常态, 意甲(中位5.30)压到4.2才是异动.
  判据= 该场a4在本联赛分布的分位 + a5/a4斜率 双确认.
- **5.3 = 排除线** (a4>5.3 真实大球仅12.8%, 直接按小球处理), 不是纳入线.

### 闸3 — 盘口+资金定形状 (档内选比分用)
- 让球盘平/平半 + 资金平和负 → 无一边倒 → 排除2-0/0-2这类单边比分, 偏对称比分(1-1/2-2).
- 明显受让 + 资金压强队 → 偏强队赢球比分.
- 进球赔率曲线峰值(最低赔率档)= 最可能总进球数, 作模数档参考(非直接抄比分).

## 三、巴黎vs阿森纳 范例(内化为prompt few-shot)
盘: 大小球线3→2.25下行, 平半, 0球10/1球4.45/2球3.5/3球4.2/4球5.8.
读: ①线下行→定档:不超3球(小球)[闸1] ②曲线峰值2球=3.5→模数2球[闸2排除大球]
    ③平半+资金平和负→无单边→对称[闸3] ④双方能进+2球位舒服→落1-1.
教训: 这是"说得出理由的众数", 不是抄最低赔率.

## 四、治 ECE 信心虚高
- confidence 必须与 tier_confidence 挂钩: 档位把握低→比分confidence强制下调.
- 非众数比分无背离理由→confidence 砍档.
- 高信心精选(S级)须三闸共振, 单闸不发S级.
