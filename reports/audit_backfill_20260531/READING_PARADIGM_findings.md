# 读盘范式重构 — 今日全量结论 (2026-05-31)

样本: 175场已配赛果/13赛日(5/06–5/28)/纯本地/生产代码零改动.

## 一、四个研究结论

### 研究B 版面主题探测 —【证伪, 不做】
chi2/df=0.99(纯随机), 版面基调分离度-0.7pp. 体彩"成版大球/小球"是事后叙事偏差.

### 研究A 两段式决策 —【成立, 病根是"定档"】
- 现状: 精确16.1% / 方向48.8% / **档位45.2% ≈ 盲猜押MID 44.6% → 模型没在读盘**
- 两段式天花板: 档判对+档内众数 = 精确28%(翻倍空间)
- 各档众数: LOW→1-0(46%) / MID→1-1(27%) / HIGH→2-2,3-1,2-3各16%(分散,弃精确比分)
- "0-0 vs 1-1纠结"=跨档错误(0-0属LOW/1-1属MID), 须先分档物理隔离, 杜绝同台比价

### 研究(赔率曲线联动) —【用户洞察证实】
a4压缩时a5/a6同步塌缩, 真实进球整体上移:
| a4 | a5中位 | a6中位 | 真实4球% | 5球% | >=6球% | >=4球% |
|---|---|---|---|---|---|---|
| <=4.2 | 5.8 | 8.8 | 31.0 | 17.2 | 3.4 | 51.7 |
| 4.2-4.8 | 7.3 | 13.0 | 22.0 | 13.6 | 1.7 | 37.3 |
| 4.8-5.3 | 8.8 | 16.0 | 10.4 | 12.5 | 6.2 | 29.2 |
| >5.3 | 11.5 | 22.0 | 2.6 | 10.3 | 0.0 | 12.8 |
→ 判大球不能看a4单点(会被单点诱盘), 要看**整条曲线塌缩**.
  曲线斜率 a5/a4<=1.3 → 66.7%大球; <=1.45 → 51.6%. 比a4单点更干净.
→ 5.3 的真正定位 = **排除线**(a4>5.3真实大球仅12.8%, 当小球处理), 不是大球纳入线.

### 研究C 相对联赛分位 —【方法学正确, 受样本量限制】
联赛a4基线天差地别: 德甲中位4.20/荷甲4.33 vs 意甲5.30/西甲5.20/解放者5.50.
→ "a4<=4.2一刀切"在德甲天天误报, 在意甲漏真异动. 用户"西甲4.2≠德甲4.2"完全成立.
- 但175场下 相对分位lift(+14.7pp) < 绝对阈值(+19.7pp): **单联赛样本太薄,分位噪声大**
- 最强=双确认: 分位<=33% 且 a4<=5.0 → 45.5%(66场)
- 诱盘量化: a4<=4.5但联赛分位>50% → 仅33.3%(单点假压)
→ **要让相对分位跑赢绝对阈值, 必须把单联赛样本做厚 = option 2 的依据**

## 二、巴黎vs阿森纳 小球读盘范例(用户口述, 须内化进prompt)
盘口: 大小球线 3→2.25(下行), 平半, 0球10 / 1球4.45 / 2球3.5 / 3球4.2 / 4球5.8.
读法链:
1. 大小球线下行 → 定档: 不超3球(小球盘) [先定档]
2. 进球曲线峰值2球=3.5(最低=最可能) → 模数档=2球
3. 平半 + 资金平和负 → 无一边倒, 排除2-0/0-2
4. 双方有进球能力 + 2球3.5舒服位 → 档内落 1-1
要点: **大小球线的"移动方向"是核心信号**(现有175样本缺开盘→当前漂移, 是数据硬伤).
      小球与大球是同一套读法反向结论, prompt必须双向对称.

## 三、prompt 重构方向(待落地)
1. 决策两段式: 先输出档位LOW/MID/HIGH+把握度, 禁止直接报比分; 再档内选比分.
2. 大球判据从"a4单点"改"曲线塌缩(a5/a4斜率)+联赛相对分位双确认"; 5.3降级为排除线.
3. 引入大小球线移动方向作一级信号(需补漂移数据源).
4. 非众数比分必须写背离理由, 否则回落众数+标低信心(治ECE虚高).
5. 0-0/1-1 跨档隔离.
